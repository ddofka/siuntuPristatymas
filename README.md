
# Parcel Delivery API

Small API for parcel, courier handling.


## Features

- Get Couriers/Parcels
- Get Courier/Parcel by ID
- Update Courier
- Update Parcel status
- Delete Parcel
- Add Courier


## Tech Stack

**Server:** Java 21, Spring Boot 3.4.4, MySQL
